# 🧊 3D Cube Puzzle

An interactive 3D cube puzzle built with CSS 3D transforms and JavaScript. Click to rotate the cube and try to match the target pattern!

## How to Play

- **Click** on the cube to rotate it 90 degrees
- Try to rotate the cube until it matches the winning pattern
- Get instant feedback on your progress
- Simple one-click interaction makes it easy to play

## Features

- **3D rotation** - Smooth CSS 3D transforms
- **Click interaction** - Simple and intuitive controls
- **Visual feedback** - Get alerts when you win or need to try again
- **Color-coded faces** - Each face has a distinct blue color scheme
- **Smooth animations** - 0.5s transition for satisfying rotation

## Technologies Used

- HTML
- CSS
- JavaScript
